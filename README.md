# 板块关系图谱分析系统 (Sector Graph)

## 📌 项目愿景
构建A股板块关系图谱，通过分析板块轮动趋势预测未来可能轮动的板块。

|------|------|------|

| V0.2 | 本地数据管理 - SQLite | 🔴 未开始 |
| V0.3 | 可视化图谱 | 🔴 未开始 |
