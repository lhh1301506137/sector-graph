### 5.2 Tab1：预测排行（首页）

用户每天打开看到的第一个页面，直接回答"哪个板块要涨？"

#### 今日摘要（顶部卡片区）

| 卡片 | 内容 |
|------|------|
| 📊 数据状态 | "已更新至 2026-02-12 15:00" / "数据待更新" |
| ✅ 昨日验证 | "昨日预测Top10中，X个今日涨幅进入实际Top20" |
| 🔥 偏差预警 | "3个关注板块累计偏差创近期新高" |
| 🆕 新板块 | "检测到2个新板块待分析" |

#### 排行表

| 列 | 说明 |
|----|------|
| 排名 | 按累计偏差得分排序 |
| ⭐ | 关注标记（可点击切换） |
| 板块名 | 板块名称 |
| 类型 | 行业/概念 |
| 今日涨幅 | 实际涨幅% |
| 预期涨幅 | 算法计算的预期% |
| 今日偏差 | 预期-实际 |
| 累计偏差 | 近30天加权累计（核心得分） |
| 趋势 | 近7天得分走势迷你图 ↗↘→ |

#### 操作
- 🔍 搜索框：输入关键词过滤
- 📋 分类筛选：行业/概念、一级/二级/三级
- ⭐ 关注切换："全部板块" / "仅关注"
- 点击行 → 展开详情（关联板块贡献明细）

### 5.3 Tab2：关系图谱

ECharts力导向图，可视化展示板块间关联。

#### 节点
- **颜色**：红色=今日涨，绿色=今日跌，深浅表示幅度
- **大小**：累计偏差得分越高越大（预测补涨目标更醒目）
- **边框**：金色边框=关注的板块
- 点击节点 → 右侧面板显示详情

#### 连线
- **粗细**：权重越大线越粗
- **颜色**：红色=正相关，绿色=负相关
- **标签**：显示关系类型

#### 右侧详情面板
- 板块基本信息（涨幅、净流入、领涨股）
- 今日预期 vs 实际偏差
- 累计偏差得分 + 近7天趋势
- 所有关联板块列表（类型、权重、锁定状态）

#### 操作
- 🔍 搜索定位（高亮匹配节点）
- 拖拽、缩放、框选
- 分类筛选（行业/概念、层级）

---

### 5.4 Tab3：数据管理

两个子面板切换：板块管理 / 关联管理

#### 板块管理
- 表格展示所有板块（名称、类型、层级、状态）
- 操作：编辑、删除、切换显示/隐藏
- 通常不需要手动添加（API自动导入），但保留手动添加能力

#### 关联管理
- 表格展示所有关联（源→目标、类型、权重、来源、锁定状态）
- 筛选：按来源（system/ai/manual）、按锁定状态
- 操作：
  - ✏️ 编辑权重
  - 🔒 锁定/解锁
  - 🗑️ 删除
  - ➕ 手动添加关联

### 5.5 设置弹窗

点击工具栏⚙️弹出，不占Tab位。

#### AI设置
| 配置项 | 说明 |
|--------|------|
| Provider | 下拉选择：DeepSeek / MiniMax / OpenAI |
| API Key | 密码输入框 |
| Base URL | 自定义接口地址（可选） |
| 模型名称 | 如 deepseek-chat |

#### 算法参数
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 时间衰减窗口 | 30天 | 偏差累计的时间范围 |
| 排行前N名 | 10 | 重点关注的板块数量 |
| 回测周期 | 60天 | 历史验证的时间跨度 |

#### 数据管理
- 导出数据（JSON备份）
- 重置数据库

---

### 5.6 工具栏操作

| 按钮 | 功能 | 耗时 | 进度显示 |
|------|------|------|---------|
| 🔄 刷新数据 | 从API拉取全板块今日涨跌，存入daily_data | 3-5秒 | 状态文字 |
| 🤖 AI分析 | 检测新板块→AI分析关联→结果进待确认 | 数分钟 | 进度条 + 可中断 |
| 📊 计算得分 | 运行偏差算法→更新排行 | 1-2秒 | 状态文字 |
| 🧪 运行回测 | 用历史数据验证预测 | 数十秒 | 进度条 |

**操作流程建议顺序**：
```
每日使用：刷新数据 → 计算得分 → 查看排行

有新板块时：刷新数据 → AI分析 → 确认结果 → 计算得分
定期验证：运行回测 → 查看准确率 → AI优化（可选）
```

### 5.8 开发优先级

#### Phase 1 — 核心（先做）
1. 后端搭建 + 全板块数据入库
2. 图谱可视化
3. 关联管理（增删改 + 锁定）
4. 偏差得分算法 + 预测排行
5. 搜索 / 筛选 / 关注

#### Phase 2 — 重要（紧接着做）
6. AI关联分析
7. 回测验证
8. 板块得分趋势图
9. 设置页面
10. 今日摘要卡片

#### Phase 3 — 增强（后续慢慢加）
11. AI参数优化
12. 数据导出/备份
13. AI分析日志
14. AI自由问答
15. 更多数据源
