## 5. 巡检执行口径
- 日常巡检：`powershell -ExecutionPolicy Bypass -File scripts/run_maintenance_mode_acceptance.ps1`
- 连续稳定性巡检：`powershell -ExecutionPolicy Bypass -File scripts/run_maintenance_mode_stability.ps1 -Runs 3 -IntervalSec 15`
- 指定地址巡检：`powershell -ExecutionPolicy Bypass -File scripts/run_maintenance_mode_stability.ps1 -BaseUrl http://127.0.0.1:<PORT> -Runs 3 -IntervalSec 15`
- Windows 一键入口：`check_maintenance_mode_stability.bat [BaseUrl] [Runs] [IntervalSec]`
